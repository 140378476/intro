\section{Overview}
\frame{\frametitle{形式}
\begin{itemize}
\item 主讲+Disucssion+Exercise+Project
\begin{itemize}
\item<1-> 主讲: 由协会对某技术主题有一定工程经验的同学进行讲解
\item<2-> Disucssion:
\begin{itemize}
\item 对该主题有兴趣且有准备的同学亦可做报告(需预先报名)
\item 全体同学提出问题进行讨论(随时)
\end{itemize}
\item<3-> Exercise: 给定简单Demo题目供大家练习
\item<4-> Project: 参与具体的Project应用相关技术
\end{itemize}
\end{itemize}
